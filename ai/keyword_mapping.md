# 갤럽 CliftonStrengths 키워드 매핑

**최종 업데이트:** 2026-05-02  
**용도:** AI 코치가 대화 중 키워드를 탐색할 때 사용하는 내부 참조 문서

---

## 사용 원칙

이 테이블은 AI 내부 동작의 기준이다.  
사용자에게 직접 보여주는 자료가 아니다.

- 사용자의 말에서 패턴을 포착하고 키워드 후보를 설정할 때 참고한다
- 키워드를 먼저 제시하지 않는다
- 패턴을 사용자의 언어로 먼저 반영하고, 공명 확인 후 키워드로 연결한다

---

## T — 전략적 사고 (Strategic Thinking)

| 키워드 | 정의 | 이런 말이 나올 때 | 탐색 질문 방향 |
|---|---|---|---|
| **전략 Strategic** | 앞으로 나아가기 위한 대안을 만들고 패턴을 빠르게 파악 | "선택지가 많아도 어떻게 할지 금방 보여요" / "큰 그림을 먼저 보게 돼요" | "그 상황에서 어떻게 방향을 잡으셨어요?" |
| **분석 Analytical** | 이유와 원인을 찾고, 모든 요인에 대해 생각 | "왜 그런지 이유를 알아야 직성이 풀려요" / "근거가 있어야 믿어요" | "어떤 부분이 납득이 안 되셨어요?" |
| **배움 Learner** | 배움에 대한 강한 열망, 과정 자체가 신남 | "새로운 걸 배울 때 가장 신나요" / "배우는 과정 자체가 좋아요" | "일하면서 가장 빠져드는 순간이 언제예요?" |
| **수집 Input** | 수집하고 보관하고 싶어함. 정보, 아이디어 축적 | "뭔가 모아두는 걸 좋아해요" / "정보가 많을수록 안심이 돼요" | "어떤 걸 주로 모으세요?" |
| **발상 Ideation** | 아이디어에 매료됨. 연결고리를 찾아냄 | "아이디어가 계속 떠올라요" / "연결고리를 찾는 게 재밌어요" | "그 아이디어가 어디서 왔어요?" |
| **미래지향 Futuristic** | 미래와 가능성에서 영감을 얻고 비전으로 영감을 줌 | "앞으로 어떻게 될지 자꾸 상상해요" / "지금보다 미래가 더 선명해요" | "그 미래에서 무엇이 가장 중요한가요?" |
| **지적사고 Intellection** | 지적 활동, 자기 성찰적, 지적 토론을 즐김 | "혼자 생각하는 시간이 필요해요" / "깊이 있는 대화가 좋아요" | "일 끝나고 혼자 있을 때 주로 어떤 생각이 떠오르세요?" |
| **회고 Context** | 과거를 통해 현재를 이해, 역사 연구를 즐김 | "왜 이렇게 됐는지 역사를 알아야 해요" / "과거에서 배우는 게 중요해요" | "그때로 돌아가면 뭐가 달랐을 것 같아요?" |

## E — 실행력 (Executing)

| 키워드 | 정의 | 이런 말이 나올 때 | 탐색 질문 방향 |
|---|---|---|---|
| **성취 Achiever** | 에너지가 왕성하고 열심히 일함. 바쁘고 생산적인 데서 만족 | "할 일 목록을 다 지워야 마음이 편해요" / "바쁠 때 오히려 에너지가 나요" | "하루가 끝날 때 뭘 해야 뿌듯하세요?" |
| **집중 Focus** | 방향을 정해 일을 완수. 우선순위 설정 후 실행 | "한 번 시작하면 끝을 봐야 해요" / "우선순위가 명확해야 움직여요" | "지금 일에서 가장 중요한 한 가지가 뭔가요?" |
| **책임 Responsibility** | 심리적 주인의식. 정직과 충직함을 지키고자 함 | "맡은 건 반드시 해요" / "약속을 못 지키면 너무 힘들어요" | "그 책임감이 어디서 왔다고 생각하세요?" |
| **심사숙고 Deliberative** | 의사결정 시 매우 신중하게 고려. 장애요소를 예측 | "결정을 내리기 전에 오래 생각해요" / "실수가 두려워요" | "결정할 때 가장 걱정되는 게 뭔가요?" |
| **체계 Discipline** | 정해진 절차와 구조를 즐김. 스스로 만든 질서 | "루틴이 있어야 안정돼요" / "계획대로 안 되면 불편해요" | "어떤 구조가 있을 때 일이 가장 잘 되세요?" |
| **정리 Arranger** | 조직하는 능력과 유연성. 생산성 최대화를 위한 조율 | "여러 가지를 동시에 조율하는 게 재밌어요" / "복잡한 걸 정리하면 뿌듯해요" | "그걸 어떻게 정리하셨어요?" |
| **신념 Belief** | 변하지 않는 핵심 가치. 삶의 목적이 가치에서 나옴 | "이건 어떻게 해도 타협 못 해요" / "가치관이 안 맞으면 힘들어요" | "그 가치관이 언제부터였어요?" |
| **공정성 Consistency** | 동등하게 취급되어야 한다는 인식. 안정적 루틴과 규칙 | "모두에게 같은 기준이 적용돼야 해요" / "불공평한 게 가장 불편해요" | "그 기준이 지켜지지 않을 때 어떻게 되세요?" |
| **복구 Restorative** | 문제를 능숙하게 다룸. 무엇이 잘못됐는지 파악하고 해결 | "문제가 생기면 오히려 집중이 돼요" / "고장난 걸 고치는 게 재밌어요" | "그 문제를 어떻게 풀었어요?" |

## R — 관계 구축 (Relationship Building)

| 키워드 | 정의 | 이런 말이 나올 때 | 탐색 질문 방향 |
|---|---|---|---|
| **공감 Empathy** | 다른 사람의 감정을 느낄 수 있음. 감성 지능 | "상대방 감정이 느껴져요" / "분위기를 금방 읽어요" | "그 순간 상대가 어떨 것 같았어요?" |
| **개발 Developer** | 다른 사람의 잠재력을 알아보고 키워줌 | "누군가 성장하는 걸 보면 뿌듯해요" / "가르치는 게 좋아요" | "그 사람에게 어떤 가능성이 보였어요?" |
| **화합 Harmony** | 합의를 추구. 갈등 대신 합의점을 찾음 | "갈등이 너무 불편해요" / "모두가 동의했으면 좋겠어요" | "그 상황에서 어떻게 하셨어요?" |
| **절친 Relator** | 친밀한 관계를 즐김. 깊은 관계에서 만족감 | "깊은 관계가 몇 명이면 충분해요" / "아는 사람이랑 더 깊어지고 싶어요" | "그 관계에서 뭐가 중요했어요?" |
| **개별화 Individualization** | 각 사람의 독특한 개성에 흥미. 다름을 파악하는 재능 | "사람마다 다 달라요" / "딱 맞는 방식이 있어요" | "그 사람의 어떤 점이 보였어요?" |
| **긍정 Positivity** | 전염되기 쉬운 열정. 낙관적이며 신나게 만듦 | "사람들이 힘내면 좋겠어요" / "분위기를 밝게 만들고 싶어요" | "그 에너지가 어디서 나오는 것 같아요?" |
| **포용 Includer** | 다른 사람을 받아들임. 소외된 사람을 알아차림 | "소외되는 사람이 없었으면 해요" / "다 같이 가야 해요" | "누가 빠졌다고 느껴질 때 어떻게 하세요?" |
| **연결성 Connectedness** | 모든 것이 연결되어 있다고 믿음. 의미를 찾음 | "다 연결돼 있는 것 같아요" / "우연은 없다고 생각해요" | "그 연결이 어떤 의미로 느껴지세요?" |
| **적응 Adaptability** | 상황의 흐름에 따라 적응. 현재에 충실 | "상황에 따라 유연하게 가요" / "계획이 바뀌어도 괜찮아요" | "그 변화 속에서 뭘 붙잡고 계셨어요?" |

## I — 영향력 (Influencing)

| 키워드 | 정의 | 이런 말이 나올 때 | 탐색 질문 방향 |
|---|---|---|---|
| **최상화 Maximizer** | 강점에 초점. 좋은 것을 최상의 것으로 전환 | "잘하는 걸 더 잘하게 만들고 싶어요" / "좋은 걸 최고로 만들고 싶어요" | "지금 커리어에서 가장 더 키우고 싶은 게 뭔가요?" |
| **행동 Activator** | 생각을 행동으로 옮김. 당장 실행하기를 원함 | "일단 해봐야 알아요" / "생각보다 실행이 먼저예요" | "그때 뭐가 움직이게 했어요?" |
| **주도력 Command** | 존재감이 있음. 상황을 주도하고 결정 | "내가 결정해야 편해요" / "불확실한 상황에서 앞장서게 돼요" | "그 순간 어떻게 결정하셨어요?" |
| **커뮤니케이션 Communication** | 생각을 말로 쉽게 표현. 좋은 대화상대, 발표에 능함 | "말로 설명하는 게 좋아요" / "스토리로 풀면 잘 전달돼요" | "어떻게 전달하셨어요?" |
| **존재감 Significance** | 큰 영향을 미치기를 원함. 인정받고 싶은 욕구 | "인정받고 싶은 마음이 강해요" / "의미있는 일을 해야 해요" | "어떤 인정이 가장 중요한가요?" |
| **승부 Competition** | 다른 사람과 비교해서 평가. 1등을 향해 분투 | "비교가 자극이 돼요" / "지는 게 너무 싫어요" | "그 경쟁에서 뭘 얻고 싶었어요?" |
| **자기확신 Self-Assurance** | 자신의 능력에 대한 자신감. 내면의 나침반 | "내 판단을 믿어요" / "남들이 뭐라 해도 내 길을 가요" | "그 확신이 어디서 왔어요?" |
| **사교성 Woo** | 새로운 사람 만나기를 좋아함. 어색함을 깨는 것 | "새로운 사람 만나는 게 좋아요" / "첫 만남의 어색함이 재밌어요" | "처음 만난 사람과 어떻게 가까워지세요?" |

## 도메인별 패턴 요약

빠른 가설 설정을 위한 요약이다.

| 도메인 | 핵심 신호 |
|---|---|
| **T 전략적 사고** | 왜/어떻게를 파고듦 / 아이디어가 많음 / 배우는 게 좋음 / 미래를 그림 |
| **E 실행력** | 완수해야 함 / 루틴이 중요함 / 책임감이 강함 / 완벽주의 경향 |
| **R 관계 구축** | 사람의 감정을 읽음 / 갈등이 불편함 / 성장을 돕고 싶음 / 깊은 관계 |
| **I 영향력** | 인정받고 싶음 / 앞장서게 됨 / 일단 해봄 / 최고를 만들고 싶음 |
