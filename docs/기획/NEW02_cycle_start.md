# NEW02. 12주 여정 시작 안내 (신규)

> 10 액션 선택 완료 직후 1회 노출. 사용자가 선택한 강점·커리어 방향·액션을 종합 요약하고 본격 12주 시작을 알림.

## 1. 화면 개요

| 항목 | 내용 |
| --- | --- |
| 화면 ID | NEW02_cycle_start |
| 페이즈 | DO → MAINTAIN 전환 |
| 역할 | 12주 시작 동기 부여 + 첫 주 액션 확정 표시 |
| 진입 경로 | 10 액션 선택 완료 직후 |
| 다음 화면 | 11 홈 (또는 NEW04 푸시 권한 요청) |

## 2. 진입 조건

- 10 완료 직후 1회 노출
- growth_cycles.status = active로 막 변경된 시점
- 재방문 시 노출되지 않음

## 3. UI 구성

- 메인 타이틀: "준비 완료! 🎉 12주 여정을 시작해요"

### 3.1 요약 카드

- 강점: Top 5 (Chip, 갤럽 34 테마)
- 커리어 방향: selected_direction
- 첫 주 액션: 선택한 action_items 리스트

### 3.2 일정 표시

- "📅 시작일: 2026.05.04 / 종료일: 2026.07.27" 표시 (today + 12주)

### 3.3 CTA

- "홈으로 가기 →"

## 4. 기능

- CTA 클릭 시 11 홈 이동 (또는 NEW04 푸시 권한 요청 화면 경유)
- 첫 주 weekly_plans는 10 완료 시 이미 생성되어 있음
- coaching_start_at 표시

## 5. 예외 처리

| 상황 | 처리 |
| --- | --- |
| weekly_plans 생성 실패 (10에서 누락) | 재시도 + 실패 시 고객센터 안내 |
| 요약 데이터 누락 | 가능한 데이터만 표시 + 안내 |

| 이벤트 | 속성 |
| --- | --- |
