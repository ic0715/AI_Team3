# NEW05. 네트워크 오류 화면 (신규)

> 서버/네트워크 연결 실패 시 노출되는 풀스크린 또는 인라인 오류 화면.

## 1. 화면 개요

| 항목 | 내용 |
| --- | --- |
| 화면 ID | NEW05_network_error |
| 페이즈 | ERROR (전역) |
| 역할 | 네트워크/서버 오류 안내 + 재시도 |
| 진입 경로 | API 5xx, fetch 실패, 타임아웃, 오프라인 |

## 2. UI 구성

- 일러스트 (와이파이 끊김 아이콘)
- 메시지: "연결이 불안정해요"
- 서브: "네트워크 상태를 확인해주세요"
- CTA: "다시 시도" (재시도 카운터 노출)
- 보조 CTA: "홈으로"

## 3. 기능

- 자동 재시도: 5초 간격, 최대 3회
- 재시도 성공 시 원래 화면으로 자동 복귀
- 오프라인 모드 지원 화면(메모, 체크 등)에서는 "오프라인" 배너만 노출하고 입력 가능 (큐 적재 후 재연결 시 자동 동기화)

## 4. 변형 케이스

| 상태 | 처리 |
| --- | --- |
| API 5xx (서버 오류) | "서비스에 일시적인 문제가 있어요" |
| API 401 (인증 만료) | 자동 로그아웃 후 02 |
| API 403 (권한 없음) | "권한이 없어요" → 홈 이동 |
| API 429 (Rate limit) | "잠시 후 다시 시도해주세요" + 대기 시간 표시 |
| 오프라인 | 오프라인 배너 + 입력은 큐 적재 |
| 타임아웃 | "응답이 늦어지고 있어요" + 재시도 |

## 5. 분석 이벤트

| 이벤트 | 속성 |
| --- | --- |
| error_occurred | error_type=network, status_code, screen, message_hash |
| error_retry_clicked | attempt_count |
| error_recovered | time_to_recover |

## 6. 접근성

- 재시도 카운터는 aria-live="polite"로 알림
- CTA는 키보드 포커스 가능, 명확한 포커스 인디케이터
