# 13. 회고 코칭 (AI 채팅)

> 주간 회고 내용을 컨텍스트로 받은 AI 코치와 대화하며 패턴·인사이트를 도출하고 다음 주 방향을 정함.

## 1. 화면 개요

| 항목 | 내용 |
| --- | --- |
| 화면 ID | 13_reflect_coaching |
| 페이즈 | MAINTAIN |
| 역할 | 회고 데이터 기반 AI 코칭 + 다음 주 방향 설계 |
| 진입 경로 | 12 주말 모드, 11 알림 카드 |

## 2. 진입 조건

- 사용자 상태 = ACTIVE
- 이번 주 회고가 작성되어 있거나, 사용자가 미작성 상태에서 직접 진입한 경우(가이드 노출)

## 3. UI 구성

### 3.1 상단 바

- 뒤로가기 → 12
- 페이지 타이틀: "회고 코칭"

### 3.2 AI 인사이트 카드 (주간 요약)

- 이번 주 핵심 패턴 요약 (AI 자동 생성)
- 강점 활용 분석
- 도전 영역

### 3.3 채팅 영역

05/08과 동일한 채팅 패턴. 자동 저장, 위기 신호 처리 모두 동일.

### 3.4 컨텍스트 안내 (인터뷰 시작 시)

- "이번 주는 X일 중 Y일 액션을 수행했어요"
- "메모에서 자주 나온 키워드: A, B, C"
- 위 데이터 기반으로 첫 질문 자동 생성

### 3.5 코어 질문 (회고 코칭용 6개)

| Q | 질문 (예시) |
| --- | --- |
| Q1 | 이번 주 액션은 7일 중 X일 수행했어요. 가장 잘했다고 느낀 날은 언제인가요? |
| Q2 | 미수행한 날의 이유를 떠올려보면? |

| Q4 | 이번 주 강점([Top1])을 어디서 활용했나요? |
| Q5 | 다음 주에 바꿔보고 싶은 점 1가지는? |
| Q6 | 다음 주 액션을 어떻게 조정하면 좋을까요? |

### 3.6 인터뷰 완료 후 정리 영역

#### 회고 요약 섹션

- 이번 주 패턴 (AI 분석)
- 강점 활용 분석
- 도전 영역

#### 다음 주 액션 제안

- 구체적인 행동 1~2개 (변경 또는 유지)
- 강점 연결 표시
- 적용 시점: "다음 월요일부터 적용돼요"

#### CTA

- Secondary: "다시 다듬기" → 인터뷰 추가
- Primary: "맞아요 (확정)" → insight_history 저장, 다음 주 weekly_actions(week_num+1) 생성

## 4. 기능

| 기능 | 동작 |
| --- | --- |
| 인터뷰 진행 | 05와 동일한 자동 저장/재개 정책 |
| 컨텍스트 자동 주입 | 이번 주 weekday_memos + weekly_actions.reflection + 강점(strength_results) + 커리어방향(career_results) + 지난주 insight_history |
| 정리 화면 전환 | 인터뷰 완료 후 자동 |
| 다시 다듬기 | 인터뷰 모드로 복귀, 추가 답변 가능 |
| 확정 | insight_history 저장 (week_num, pattern, action, strength_link, mode=retro) + 다음 주 weekly_actions 생성 → 11 이동 |

## 5. AI 시스템 프롬프트

- **코칭 모드**: RETRO (회고 코칭) — 단일 흐름: 한 주 인상 → 실행 경험 → 잘 된 부분 → 효용 평가 → 변경 의사 → 강점 매칭 → 본인 표현으로 굳히기
- **목적**: 이미 진행 중인 액션의 효용 평가 + 다음 주 액션 조정 (※ 새 주제 발굴 X)
- **컨텍스트**: 주간 메모+강점+커리어 방향을 받아 패턴·인사이트 도출 및 다음 주 방향 제안
- **MCC 코칭 원칙** + **갤럽 강점 프레임** 유지
- **시간 정책**: 다음 주 월요일부터 적용 — 이번 주 데이터는 그대로 보존되어 12주 여정의 한 페이지로 남음
- **저장**: coaching_sessions에 messages JSONB로 대화 원문 저장
- 인사이트는 insight_history에 저장 (본문은 저장하지 않고 요약 키워드만, 원문은 coaching_sessions에 보관)
- 위기 신호 / PII 처리는 05와 동일 정책 적용

## 6. 예외 처리

| 상황 | 처리 |
| --- | --- |
| 회고 미작성 상태 진입 | "한 주 회고를 먼저 작성해주세요" 가이드 + 12로 |
| 인터뷰 중 새로고침 | 재개 정책에 따라 복원 |
| 정리 분석 실패 | "분석에 실패했어요. 다시 시도하시겠어요?" |
| 다음 주 plan 생성 실패 | 트랜잭션 롤백, 재시도 |
| 확정 후 변경 요청 | 1회 한정 24시간 내 수정 가능 |
| 위기 신호 | 즉시 중단 + 1393 안내 |
