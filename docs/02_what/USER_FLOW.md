# 사용자 흐름 (User Flow)

**최종 업데이트:** 2026-05-02  
**상태:** 확정 (ADR_001 v2 기반)

---

## 전체 흐름

```
[1단계] 회원가입
    ↓
[2단계] 기본설문 (3개 항목)
    ↓
[3단계] 파일 업로드 화면
    ├── 파일 있음 → 업로드
    └── 파일 없음 → "간단한 인터뷰로 대체할게요" 선택
                        ↓
                   [3-B] 보완 인터뷰 (5개 질문)
    ↓
[4단계] 첫 번째 코칭 세션 ← 진짜 코칭 시작
    ↓
[5단계] 세션 마무리 (키워드 선택 + 첫 행동)
    ↓
[6단계] 메인 앱 (이후 세션 누적)
```

---

## 1단계 — 회원가입

- 이메일 / 비밀번호
- 구글 소셜 로그인
- 이름 (닉네임)

---

## 2단계 — 기본설문

코칭에 꼭 필요한 최소한의 맥락만 수집한다.
3개 항목, 모두 자유 입력이다.

```
1. 어떻게 불러드릴까요?
   [이름 또는 닉네임 입력]

2. 현재 어떤 일을 하고 계세요?
   [한 줄 자유 입력 / 예: PM, 마케터, 개발자...]

3. 요즘 일이나 커리어에서 가장 마음에 걸리는 게 뭔가요?
   [한 줄 자유 입력]
```

3번 항목은 첫 코칭 세션의 오프닝 질문과 자연스럽게 연결된다.

**폐기된 항목 (기존 대비)**
- 나이
- 소속/회사
- 연차
- 감정 선택지
- 관심사/해결하고 싶은 문제 (별도 항목)

---

## 3단계 — 파일 업로드 화면

### 화면 구성

```
┌─────────────────────────────────────────┐
│  강점 리포트나 이력서가 있으면           │
│  업로드해주세요                          │
│                                         │
│  업로드 가능한 자료                      │
│  · 갤럽 CliftonStrengths 결과지         │
│  · 이력서                               │
│  · MBTI, 버크만 등 기타 진단 결과       │
│                                         │
│        [📄 파일 업로드]                 │
│                                         │
│  ─────────────────────────────────────  │
│                                         │
│  따로 파일이 없어요.                    │
│  간단한 인터뷰로 대체할게요.  →         │
│                                         │
└─────────────────────────────────────────┘
```

### 파일이 있는 경우

업로드 완료 → 바로 4단계(첫 코칭 세션)로 이동

업로드된 파일은 코칭 대화의 배경 맥락으로만 활용한다.
AI가 먼저 분석 결과를 제시하지 않는다.

### 파일이 없는 경우

"간단한 인터뷰로 대체할게요" 선택 → 3-B 보완 인터뷰로 이동

---

## 3-B단계 — 보완 인터뷰 (파일 없는 사용자)

갤럽 강점 리포트를 대체하는 5개 질문이다.
채팅 형식으로 하나씩 진행한다.

### Q1 — 에너지 원천
```
일하면서 시간 가는 줄 모르고 몰입했던 순간이 있었나요?
어떤 상황이었는지 떠오르는 대로 말씀해주세요.
```
→ 탐색: 어떤 활동에서 에너지가 올라오는가 (배움/발상/성취/행동 등)

### Q2 — 타인이 인식하는 강점
```
주변 사람들이 나에게 자주 부탁하거나
"이건 네가 잘하잖아"라고 말하는 게 있나요?
```
→ 탐색: 자신이 잘 인식 못 하는 강점 (갤럽의 "기여하는 것"에 해당)

### Q3 — 핵심 가치
```
일할 때 절대 타협하기 싫은 것,
혹은 이것만큼은 지키고 싶다는 게 있나요?
```
→ 탐색: 신념/책임/공정성/연결성 등 가치 기반 키워드

### Q4 — 관계/역할 스타일
```
팀이나 사람들 사이에서 내가 자연스럽게 맡게 되는 역할이 있나요?
예를 들어, 분위기를 만드는 사람, 정리해주는 사람,
앞장서는 사람, 깊이 들어주는 사람 같은 것이요.
```
→ 탐색: 관계 구축 / 영향력 도메인 스타일

### Q5 — 커리어 맥락 연결
```
지금 커리어에서 가장 해결하고 싶은 것,
또는 가장 마음에 걸리는 게 뭔가요?
```
→ 첫 코칭 세션 오프닝과 연결되는 다리 역할

인터뷰 완료 후 AI 내부적으로 갤럽 키워드 후보 3~5개 잠정 설정.
→ 4단계(첫 코칭 세션)로 이동

---

## 4단계 — 첫 번째 코칭 세션

### 오프닝
```
"요즘 일이나 커리어에서 가장 마음에 걸리는 게 뭔가요?"
```
기본설문 3번 항목과 연결해서 자연스럽게 시작한다.

### 대화 진행
- AI가 사용자의 답변에서 에너지가 실린 단어를 포착한다
- MCC 수준의 열린 질문으로 탐색을 이어간다
- 한 번에 하나의 질문만 한다
- 4~6턴 후 패턴이 드러나면 반영한다

### 갤럽 키워드 반영 순서
```
1. 패턴을 사용자의 언어로 먼저 반영
   "말씀하신 것들에서 [패턴]이 느껴지는데, 어떻게 느껴지세요?"

2. 사용자 공명 확인 후 키워드 연결
   "갤럽 강점으로 보면 [키워드]와 많이 닿아 있어요.
    이 단어가 와닿으세요?"
```

---

## 5단계 — 세션 마무리

```
AI: "오늘 대화에서 어떤 걸 가져가실 것 같아요?"
    ↓
사용자 자유 답변
    ↓
AI: 키워드 1~3개 제시
    ↓
사용자가 직접 선택/수정
    ↓
AI: "오늘 커리어에서 딱 하나만 해본다면 뭘 해보고 싶으세요?"
    ↓
사용자가 첫 번째 행동 직접 정함
```

---

## 6단계 — 메인 앱

### 홈
- 내 키워드 (누적)
- 다음 코칭 세션 시작
- 체크인 현황

### 이후 코칭 세션
- 이전 키워드를 맥락으로 누적
- 새로운 패턴 탐색 계속

### 체크인 알림 (이틀 단위)
```
"오늘 일하면서 어떤 순간이 기억에 남으세요?"
```

### 회고
- Keep / Problem / Try
- AI가 키워드와 연결되는 패턴 반영

---

## 이전 구조와 비교

### 이전 구조 (폐기)
```
회원가입 → 기본설문(상세) → 파일업로드
→ AI 인터뷰 10개 → AI 진단 리포트
(종합요약 / 강점개선 / 키워드 / 로드맵 / 역량갭)
→ 코칭 세션
```

### 새로운 구조
```
회원가입 → 기본설문(3개) → 파일업로드 or 보완인터뷰(5개)
→ 첫 코칭 세션 (MCC 수준)
→ 사용자가 직접 키워드 선택
```

핵심 차이: AI가 결과를 주는 구조 → 사용자가 발견하는 구조
