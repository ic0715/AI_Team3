# 코칭 대화 설계

**최종 업데이트:** 2026-05-02  
**상태:** 확정 (ADR_001 v2 기반)  
**기반:** ICF MCC 핵심역량 + 갤럽 CliftonStrengths 34

---

## 핵심 원칙

> "온보딩이 끝나는 순간 코칭이 시작된다"

기본설문 + 파일업로드(또는 보완인터뷰)로 강점 맥락을 확보한 뒤,
첫 코칭 세션부터 MCC 수준의 코칭이 진행된다.

---

## 온보딩 단계별 코칭 연결

### 기본설문 3번 항목 → 오프닝 연결

```
기본설문:
"요즘 일이나 커리어에서 가장 마음에 걸리는 게 뭔가요?"
    ↓
첫 코칭 오프닝:
"[기본설문 답변 내용]에 대해 좀 더 이야기해주실 수 있어요?"
또는
"요즘 일이나 커리어에서 가장 마음에 걸리는 게 뭔가요?"
```

### 보완 인터뷰 → 코칭 배경 맥락

파일이 없는 사용자의 보완 인터뷰 5개 답변은
AI가 내부적으로 분석해서 갤럽 키워드 후보 3~5개를 잠정 설정한다.
이 정보는 코칭 대화의 배경 맥락으로만 활용한다.
사용자에게 먼저 제시하지 않는다.

### 파일 업로드 → 코칭 배경 맥락

업로드된 갤럽 결과지나 이력서 정보도 동일하게 처리한다.
대화 중 자연스럽게 연결할 때만 참고한다.

```
❌ "갤럽 결과에서 전략이 1위시네요."
✅ "아까 말씀하신 것처럼 큰 그림을 먼저 보는 경향이 있으신 것 같아요.
   갤럽 결과에서도 전략이 상위에 있으시더라고요.
   이게 어떻게 느껴지세요?"
```

## 보완 인터뷰 설계

파일이 없는 사용자를 위한 5개 질문이다.
채팅 형식, 하나씩 순서대로 진행한다.

### Q1 — 에너지 원천 탐색
*탐색 키워드: 배움, 발상, 성취, 행동, 사교성 등*

```
일하면서 시간 가는 줄 모르고 몰입했던 순간이 있었나요?
어떤 상황이었는지 떠오르는 대로 말씀해주세요.
```

### Q2 — 타인이 인식하는 강점
*탐색 키워드: 전략, 분석, 정리, 공감, 개별화, 커뮤니케이션 등*

```
주변 사람들이 나에게 자주 부탁하거나
"이건 네가 잘하잖아"라고 말하는 게 있나요?
```

### Q3 — 핵심 가치 / 신념
*탐색 키워드: 신념, 책임, 공정성, 연결성, 화합 등*

```
일할 때 절대 타협하기 싫은 것,
혹은 이것만큼은 지키고 싶다는 게 있나요?
```

### Q4 — 관계 / 역할 스타일
*탐색 키워드: 공감, 개발, 주도력, 커뮤니케이션, 절친, 긍정 등*

```
팀이나 사람들 사이에서 내가 자연스럽게 맡게 되는 역할이 있나요?
예를 들어, 분위기를 만드는 사람, 정리해주는 사람,
앞장서는 사람, 깊이 들어주는 사람 같은 것이요.
```

### Q5 — 커리어 맥락 연결
*첫 코칭 세션 오프닝과 연결*

```
지금 커리어에서 가장 해결하고 싶은 것,
또는 가장 마음에 걸리는 게 뭔가요?
```

## AI 내부 동작 원리

사용자에게 보이지 않는 부분이다.

```
1. 보완 인터뷰 또는 파일 업로드 정보를 배경 맥락으로 설정
2. 코칭 대화 시작
3. 답변에서 에너지가 실린 단어 포착
4. 도메인 가설 설정 (T/E/R/I 중 어디?)
5. 갤럽 키워드 후보 2~3개 잠정 선정
6. 그 키워드를 드러낼 다음 질문 생성
7. 사용자 답변에 따라 가설 수정 반복
```

키워드는 사용자에게 바로 제시하지 않는다.
4~6턴 대화 후 패턴이 충분히 드러났을 때 반영한다.

## 갤럽 키워드 매핑 테이블

### T — 전략적 사고 (Strategic Thinking)

| 키워드 | 이런 말이 나올 때 | 탐색 질문 방향 |
|---|---|---|
| **전략 Strategic** | "선택지가 많아도 어떻게 할지 금방 보여요" / "큰 그림을 먼저 보게 돼요" | "그 상황에서 어떻게 방향을 잡으셨어요?" |
| **분석 Analytical** | "왜 그런지 이유를 알아야 직성이 풀려요" / "근거가 있어야 믿어요" | "어떤 부분이 납득이 안 되셨어요?" |
| **배움 Learner** | "새로운 걸 배울 때 가장 신나요" / "배우는 과정 자체가 좋아요" | "일하면서 가장 빠져드는 순간이 언제예요?" |
| **수집 Input** | "뭔가 모아두는 걸 좋아해요" / "정보가 많을수록 안심이 돼요" | "어떤 걸 주로 모으세요?" |
| **발상 Ideation** | "아이디어가 계속 떠올라요" / "연결고리를 찾는 게 재밌어요" | "그 아이디어가 어디서 왔어요?" |
| **미래지향 Futuristic** | "앞으로 어떻게 될지 자꾸 상상해요" / "지금보다 미래가 더 선명해요" | "그 미래에서 무엇이 가장 중요한가요?" |
| **지적사고 Intellection** | "혼자 생각하는 시간이 필요해요" / "깊이 있는 대화가 좋아요" | "일 끝나고 혼자 있을 때 주로 어떤 생각이 떠오르세요?" |
| **회고 Context** | "왜 이렇게 됐는지 역사를 알아야 해요" / "과거에서 배우는 게 중요해요" | "그때로 돌아가면 뭐가 달랐을 것 같아요?" |

### E — 실행력 (Executing)

| 키워드 | 이런 말이 나올 때 | 탐색 질문 방향 |
|---|---|---|
| **성취 Achiever** | "할 일 목록을 다 지워야 마음이 편해요" / "바쁠 때 오히려 에너지가 나요" | "하루가 끝날 때 뭘 해야 뿌듯하세요?" |
| **집중 Focus** | "한 번 시작하면 끝을 봐야 해요" / "우선순위가 명확해야 움직여요" | "지금 일에서 가장 중요한 한 가지가 뭔가요?" |
| **책임 Responsibility** | "맡은 건 반드시 해요" / "약속을 못 지키면 너무 힘들어요" | "그 책임감이 어디서 왔다고 생각하세요?" |
| **심사숙고 Deliberative** | "결정을 내리기 전에 오래 생각해요" / "실수가 두려워요" | "결정할 때 가장 걱정되는 게 뭔가요?" |
| **체계 Discipline** | "루틴이 있어야 안정돼요" / "계획대로 안 되면 불편해요" | "어떤 구조가 있을 때 일이 가장 잘 되세요?" |
| **정리 Arranger** | "여러 가지를 동시에 조율하는 게 재밌어요" / "복잡한 걸 정리하면 뿌듯해요" | "그걸 어떻게 정리하셨어요?" |
| **신념 Belief** | "이건 어떻게 해도 타협 못 해요" / "가치관이 안 맞으면 힘들어요" | "그 가치관이 언제부터였어요?" |
| **공정성 Consistency** | "모두에게 같은 기준이 적용돼야 해요" / "불공평한 게 가장 불편해요" | "그 기준이 지켜지지 않을 때 어떻게 되세요?" |
| **복구 Restorative** | "문제가 생기면 오히려 집중이 돼요" / "고장난 걸 고치는 게 재밌어요" | "그 문제를 어떻게 풀었어요?" |

### R — 관계 구축 (Relationship Building)

| 키워드 | 이런 말이 나올 때 | 탐색 질문 방향 |
|---|---|---|
| **공감 Empathy** | "상대방 감정이 느껴져요" / "분위기를 금방 읽어요" | "그 순간 상대가 어떨 것 같았어요?" |
| **개발 Developer** | "누군가 성장하는 걸 보면 뿌듯해요" / "가르치는 게 좋아요" | "그 사람에게 어떤 가능성이 보였어요?" |
| **화합 Harmony** | "갈등이 너무 불편해요" / "모두가 동의했으면 좋겠어요" | "그 상황에서 어떻게 하셨어요?" |
| **절친 Relator** | "깊은 관계가 몇 명이면 충분해요" / "아는 사람이랑 더 깊어지고 싶어요" | "그 관계에서 뭐가 중요했어요?" |
| **개별화 Individualization** | "사람마다 다 달라요" / "딱 맞는 방식이 있어요" | "그 사람의 어떤 점이 보였어요?" |
| **긍정 Positivity** | "사람들이 힘내면 좋겠어요" / "분위기를 밝게 만들고 싶어요" | "그 에너지가 어디서 나오는 것 같아요?" |
| **포용 Includer** | "소외되는 사람이 없었으면 해요" / "다 같이 가야 해요" | "누가 빠졌다고 느껴질 때 어떻게 하세요?" |
| **연결성 Connectedness** | "다 연결돼 있는 것 같아요" / "우연은 없다고 생각해요" | "그 연결이 어떤 의미로 느껴지세요?" |
| **적응 Adaptability** | "상황에 따라 유연하게 가요" / "계획이 바뀌어도 괜찮아요" | "그 변화 속에서 뭘 붙잡고 계셨어요?" |

### I — 영향력 (Influencing)

| 키워드 | 이런 말이 나올 때 | 탐색 질문 방향 |
|---|---|---|
| **최상화 Maximizer** | "잘하는 걸 더 잘하게 만들고 싶어요" / "좋은 걸 최고로 만들고 싶어요" | "지금 커리어에서 가장 더 키우고 싶은 게 뭔가요?" |
| **행동 Activator** | "일단 해봐야 알아요" / "생각보다 실행이 먼저예요" | "그때 뭐가 움직이게 했어요?" |
| **주도력 Command** | "내가 결정해야 편해요" / "불확실한 상황에서 앞장서게 돼요" | "그 순간 어떻게 결정하셨어요?" |
| **커뮤니케이션 Communication** | "말로 설명하는 게 좋아요" / "스토리로 풀면 잘 전달돼요" | "어떻게 전달하셨어요?" |
| **존재감 Significance** | "인정받고 싶은 마음이 강해요" / "의미있는 일을 해야 해요" | "어떤 인정이 가장 중요한가요?" |
| **승부 Competition** | "비교가 자극이 돼요" / "지는 게 너무 싫어요" | "그 경쟁에서 뭘 얻고 싶었어요?" |
| **자기확신 Self-Assurance** | "내 판단을 믿어요" / "남들이 뭐라 해도 내 길을 가요" | "그 확신이 어디서 왔어요?" |
| **사교성 Woo** | "새로운 사람 만나는 게 좋아요" / "첫 만남의 어색함이 재밌어요" | "처음 만난 사람과 어떻게 가까워지세요?" |

## 키워드 반영 방식

**1단계: 이름 없이 패턴 먼저 반영**

```
AI: "말씀을 들으면서 느낀 건데요.
    [사용자가 보여준 패턴을 사용자의 언어로].
    이게 어떻게 느껴지세요?"
```

**2단계: 사용자 공명 확인 후 키워드 연결**

```
AI: "갤럽 강점으로 보면 [키워드]와 많이 닿아 있어요.
    이 단어가 와닿으세요?"
```

**3단계: 세션 마무리 — 사용자가 직접 선택**

```
AI: "오늘 대화에서 어떤 걸 가져가실 것 같아요?"
→ 사용자 자유 답변
→ AI: "[키워드 1~3개] 중에서 지금 가장 맞는 것 같은 게 뭔가요?"
→ 사용자 직접 선택/수정
→ AI: "오늘 커리어에서 딱 하나만 해본다면 뭘 해보고 싶으세요?"
→ 사용자가 첫 번째 행동 직접 정함
```

## 대화 흐름 예시

### 케이스 A — 성취/집중 패턴

```
AI: 요즘 일이나 커리어에서 가장 마음에 걸리는 게 뭔가요?

사용자: 열심히 살고 있는데 뭔가 방향이 없는 것 같아서요.

AI: "열심히 살고 있다"고 하셨는데,
    요즘 하루가 끝날 때 어떤 느낌이 드세요?

사용자: 뭔가 많이 했는데 정작 중요한 건 못 한 것 같아요.

AI: 그 "중요한 것"이 어떤 건지,
    지금 떠오르는 게 있으세요?
```
→ AI 내부: 성취(Achiever) + 집중(Focus) 가설

### 케이스 B — 관계/존재감 패턴

```
AI: 요즘 일이나 커리어에서 가장 마음에 걸리는 게 뭔가요?

사용자: 팀에서 제 역할이 뭔지 모르겠어요.
        열심히 하는데 인정받는 느낌이 없어요.

AI: "인정받는 느낌이 없다"고 하셨는데,
    어떤 순간에 그게 가장 강하게 느껴지세요?

사용자: 제가 도와줬는데 그냥 당연하게 여기는 것 같을 때요.

AI: 그 도움이 상대에게 어떻게 전달됐으면 좋겠으셨어요?
```
→ AI 내부: 공감(Empathy) + 존재감(Significance) 가설

### 케이스 C — 발상/미래지향 패턴

```
AI: 요즘 일이나 커리어에서 가장 마음에 걸리는 게 뭔가요?

사용자: 하고 싶은 게 너무 많아서 뭘 해야 할지 모르겠어요.

AI: 하고 싶은 것들 중에서
    요즘 가장 자주 머릿속에 떠오르는 게 뭔가요?

사용자: 결국 제 콘텐츠를 만들고 싶은 것 같아요.

AI: "제 콘텐츠"라고 하셨는데,
    어떤 이야기를 하고 싶으세요?
```
→ AI 내부: 발상(Ideation) + 미래지향(Futuristic) 가설
