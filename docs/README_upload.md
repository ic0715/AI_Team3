---

##  핵심 원칙

### 1. 해석하지 않는다
```
✅ DO: 갤럽 PDF → "전략, 분석, 배움" (이름만 추출)
❌ DON'T: "전략 강점이 높으므로 기획자에 적합"
```

### 2. 판단하지 않는다
```
✅ DO: 이력서 → "PM, 3년" (팩트만 추출)
❌ DON'T: "리더십 8점, 증거 5개로 검증됨"
```

### 3. 미리 준비하지 않는다
```
✅ DO: Phase 2에 데이터 전달
❌ DON'T: "이런 질문을 할 예정입니다"
```

## 📊 v1.0 → v2.0 주요 변경사항

| 항목 | v1.0 | v2.0 |
|------|------|------|
| **역할** | AI 분석가 | 재료 준비자 |
| **소프트스킬 추론** | ✅ 포함 | ❌ 삭제 |

| **이력서 업로드** | 필수 | 선택 |
| **갤럽 강점** | Top 5 | Top 10 |
| **UI 화면** | 6개 | 4개 |
| **코드량** | ~500줄 | ~150줄 |

##  전체 플로우

```
[Phase 1] 기본 정보 입력
   ↓
[S2] 파일 업로드
│
├─> 갤럽 PDF 업로드
│   └─> Top 10 강점 추출 (해석 없음)
│
├─> 이력서 업로드 (선택)
│   └─> 경력 팩트만 추출
│
└─> 또는 직접 입력
    └─> 텍스트로 입력
   ↓
Phase 2에 데이터 전달
   ↓
[Phase 2] AI 코칭 인터뷰
   └─> S2 데이터를 컨텍스트로 활용
   └─> AI가 대화 중 자연스럽게 연결
```

##  S2 → Phase 2 전달 데이터

```typescript
interface S2ToPhase2Data {
  // Phase 1 기본 정보
  user_info: {
    name: string;
    current_role: string;
    concern_draft: string;
  };
  
  // S2 추출 데이터
  gallup?: {
    top10: Array<{
      rank: number;
      name_ko: string;
      name_en: string;
      category: string;  // "전략적 사고" | "실행력" | "관계 구축" | "영향력"
    }>;
  };
  
  career?: {
    current_company: string;
    facts: Array<{
      company: string;
      position: string;
      period: string;
    }>;
  };
  
  // 메타
  meta: {
    s2_completed_at: string;
    has_gallup: boolean;
    has_career: boolean;
  };
}
```

##  성공 지표

| 지표 | 목표 | 측정 방법 |
|------|------|----------|
| **갤럽 파싱 성공률** | > 90% | PDF 업로드 → Top 10 추출 완료 |
| **파싱 소요 시간** | < 3초 | 업로드 → 확인 화면 |
| **Phase 2 진입률** | > 80% | S2 완료 → Phase 2 시작 |
| **사용자 이탈률** | < 20% | S2 중 이탈 |

##  차별화 포인트

### 경쟁사 vs 우리

```
┌─────────────────────────────────────┐
│  일반 자기계발 앱       우리         │
├─────────────────────────────────────┤
│  AI가 강점 분석   →  추출만 (해석X) │
│  AI가 진로 추천   →  사용자가 탐색  │
│  결과 리포트      →  대화로 발견    │

```

##  갤럽 코치 검증 포인트

Phase 2 연동 시 갤럽 코치가 확인할 체크리스트:

- [ ] AI가 강점을 먼저 해석하는가?
- [ ] 강점을 라벨링 도구로 쓰는가?
- [ ] 대화 흐름보다 강점을 우선하는가?
- [ ] 사용자 거부 시 즉시 철회하는가?

**검증자**: 갤럽 강점코치 2명

##  UI 플로우 (4개 화면)

```
1. [업로드]
   - 파일 선택 or 직접 입력
   - 건너뛰기 가능

2. [파싱 중]
   - 프로그레스 바
   - "강점 정보 확인 중..."

3. [결과 확인]
   - Top 10 강점 리스트
   - 카테고리 표시 (T/E/R/I)

4. [확인 완료]
   - ✓ 확인한 정보
   - 다음 단계 안내
   - [다음: AI 코치 만나기 →]
```
