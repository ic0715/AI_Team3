# 타겟 사용자 정의

---

## 핵심 타겟: C 유형

> 뭘 해야 할지 자체가 모호한 직장인.  
> 방향이 없어서 시작을 못 하는 사람.

---

## 사용자 유형 분류

직장인 중 자기계발/커리어 고민이 있는 사람을 세 유형으로 나눌 수 있다.

### A 유형 — 실행력 부족
머리로는 아는데 몸이 안 따라가는 사람.  
방향은 이미 알고 있다. 실행이 안 될 뿐이다.

→ 필요한 것: 습관 설계, accountability  
→ 다른 앱들도 잘 해결한다

### B 유형 — 우선순위 모호
뭘 먼저 해야 할지 막연한 사람.  
방향은 대충 알지만 순서를 모르는 상태다.

→ 필요한 것: 구조화된 질문, 로드맵  
→ 다른 앱들도 잘 해결한다

### C 유형 — 방향 자체가 없음 ← **우리의 핵심 타겟**
뭘 해야 할지 자체가 모호한 사람.  
자신이 뭘 원하는지조차 불분명하다.

→ 필요한 것: 진짜 코칭, 자기발견의 경험  
→ **다른 앱들이 해결하지 못한다**

---

## C 유형의 구체적인 모습

**인구통계**
- 직장인 (직무, 연차 무관)
- 자기계발이나 미래에 대한 고민과 의지가 있는 사람

**심리적 상태**
- "열심히 살고 있는데 뭔가 방향이 없는 것 같아요"
- "뭘 해야 할지는 아는데 시작을 못 하겠어요"
- "솔직히 내가 뭘 원하는지 모르겠어요"
- "이직을 생각하는데 뭘 바꿔야 할지 모르겠어요"
- "열심히 하는데 인정받는 느낌이 없어요"

**행동 패턴**
- 자기계발 콘텐츠를 소비하지만 실행으로 이어지지 않는다
- 목표를 세웠다가 방향이 맞는지 확신이 없어 포기한다
- 혼자서는 정리가 안 된다
- 코치나 멘토가 있으면 좋겠다는 생각을 한다

---

## C 유형이 우리 서비스를 통해 경험하는 것

세션 전: "내가 뭘 원하는지 모르겠어"

세션 후: "대화하다 보니 내가 어떤 사람인지 알게 됐어"

이 경험이 이 서비스의 핵심이다.

---

## A, B 유형은?

A와 B 유형도 이 서비스를 쓸 수 있다.  
그러나 이들은 C 유형을 거친 후에  
자연스럽게 서비스 안에 머무르는 구조다.

C → 발견 → 방향 설정 → A/B 유형의 실행 지원

## 타겟이 아닌 사람

- 이미 방향이 명확하고 실행 도구만 필요한 사람
- 심리적 치유나 상담이 필요한 사람
- 재정, 의료, 법률 조언이 필요한 사람
