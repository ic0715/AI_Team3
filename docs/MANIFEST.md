# Project Manifest
 **나만의 성장을 설계하고, 완주까지 함께하는 당신의 페이스메이커**<br>
본 문서는 프로젝트의 핵심 철학, 운영 원칙 및 구조를 정의하고 있습니다.

---

## Mission Statement
우리는 자기계발을 희망하지만 실천에 어려움을 겪는 현대 직장인들에게 관성적인 삶에서 벗어나 삶의 비전과 커리어의 방향을 제시하고자 합니다. 오늘의 실천이 내일의 비전으로 연결되는 가시적인 경로를 제공하여, 단순한 'To-do' 관리가 아니라 '나의 하루에 의미를 부여하는' 성장의 여정을 돕는 것을 미션으로 합니다.

---

## Core Values 

우리는 사용자가 '작심삼일'의 굴레를 벗어나 지속 가능한 성장을 이룰 수 있도록 다음 세 가지 가치에 집중합니다.

1. **성장 (Growth)**: 단순한 수치 달성을 넘어 '미래를 위한 빌드업'으로 재정의하여 어제보다 나은 나를 발견하는 체감형 경험을 제공합니다.
2. **방향 발견 (Direction)**: 강점 검사 및 커리어 질문 등 객관적 데이터를 통해 나를 먼저 이해하고, 나에게 맞는 목적지를 설정하도록 돕습니다.
3. **목표 설정 (Goal Setting)**: Have, Be, Do를 바탕으로 막연한 꿈을 실행 가능한 구체적 단계(Milestone)로 정제하여 실질적인 변화를 이끌어냅니다.

---

## Operating Principles 

모든 기능 구현과 디자인은 아래의 3대 원칙을 최우선으로 고려합니다.

**1. 시스템 기반 (System over Willpower)**<br>
인간의 의지력은 유한합니다. 사용자의 의지에만 의존하지 않고도 습관이 형성될 수 있도록 **환경 세팅**과 **페이스 메이커** 시스템을 구축합니다.

**2. 데일리 루틴 (Daily Engagement)**<br>
사용자의 일상에 자연스럽게 스며들어야 합니다. 접속 자체가 부담이 되지 않도록 **가벼운 접근성**과 **심리적 문턱을 낮춘 UI**를 유지합니다.

**3. 즉각적 피드백 (Easy & Fun UX)**<br>
자기계발의 지루함을 깨기 위해 작은 실천에도 즉각적인 보상과 시각적 피드백을 제공하여 사용자의 흥미를 지속적으로 유지합니다.

**4. 데이터 기반의 맞춤형 연결 (Data-Driven Insight)**<br>
AI는 사용자의 정보와 성향, 실적을 분석하여 최적의 방향을 제안하고, 이탈 방지를 위한 맞춤형 추천 기능을 통해 지속적인 동기를 부여합니다.
