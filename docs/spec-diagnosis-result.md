# spec-diagnosis-result.md — 진단 결과 화면 기능 명세

## 개요

CareerPT 온보딩 플로우의 마지막 단계(Step 4 / 4)입니다.
기본 설문, 내 정보 업로드(이력서·강점 진단 결과), 음성 인터뷰 결과를 AI가 종합 분석하여
현황 진단 결과와 커리어 키워드·목표·역량 제안을 제공합니다.
사용자는 AI 제안을 검토하고 직접 수정한 뒤 확정합니다.

- 입력: 기본 설문 응답, 업로드 파일(이력서·강점 진단 결과), 인터뷰 음성 텍스트
- 출력: 종합 요약, 강점/개선 영역, 갤럽 34 강점 기반 키워드 5개, 목표 로드맵, 역량 GAP

## 기능 1 — 로딩 화면 (AI 분석 진행 표시)

### 목적
실제 AI 처리 대기를 위한 필수 화면이다. 이력서(PDF/이미지), 강점 진단 결과, 인터뷰 전체 텍스트를 동시에 AI에 전달하면 컨텍스트가 매우 크기 때문에 실제 응답 생성까지 수 초~수십 초가 소요된다. 단계별 텍스트 표시는 사용자가 막연히 기다리지 않도록 진행 상황을 인지시키는 역할을 겸한다.

### 진입 조건
인터뷰(Step 3) 완료 직후 자동 진입

### 표시 내용
스피너 + 3단계 분석 진행 상태를 순차적으로 표시

| 순서 | 표시 텍스트 | 전환 간격 |
|---|---|---|
| 1 | 기본 정보 및 자료 분석 중 | 550ms |
| 2 | 인터뷰 내용 분석 중 | 550ms |
| 3 | 진단 결과 도출 중 | 550ms → 결과 전환 |

### 단계별 상태 스타일
| 상태 | 점 색상 | 텍스트 |
|---|---|---|
| 대기 | 회색 (`#E4E4DF`) | 회색 muted |
| 진행 중 | 파랑 (pulse 애니메이션) | 파란 bold |
| 완료 | 초록 (`#16A34A`) | 검정 |

### 동작 흐름
```
로딩 화면 진입
  → 3단계 순차 표시
  → AI 응답 수신 완료
  → 로딩 화면 숨김
  → 결과 화면 표시 + AI 추천 키워드 자동 선택
```

## 기능 2 — 종합 요약

### 표시 내용
- AI가 설문·업로드 자료·인터뷰를 종합해 생성한 1~2문장 텍스트
- 현재 상태를 나타내는 키워드 배지 (읽기 전용)

### 키워드 배지 색상
- 배경: `#1A1A18` (검정)
- 텍스트: `#FFFFFF` (흰색)
- 모든 배지 동일 스타일 적용 (색상 구분 없음)

> 수정 불가, 읽기 전용. AI 제안 섹션의 키워드(기능 4)와 별개.

## 기능 3 — 강점 / 개선 영역

### 표시 내용
2열 카드로 강점과 개선 영역을 나란히 표시

| 항목 | 배경색 | 제목색 | 텍스트색 |
|---|---|---|---|
| 강점 | `#E0F4FF` (하늘색) | `#0369A1` | `#0C4A6E` |
| 개선 영역 | `#FFE4EF` (분홍색) | `#BE185D` | `#831843` |

- 각 항목 2~3개 세부 내용으로 구성

### 동작 흐름
```
뷰 모드 (기본)
  → [수정] 클릭
  → 강점 항목 / 개선 영역 항목 각각 textarea 편집 가능
  → [완료] 클릭
  → 뷰 모드로 전환 (수정 내용 반영)
```

## 기능 4 — AI 제안: 나를 정의하는 키워드

### 뷰 모드 (기본)

- AI가 갤럽 CliftonStrengths 34개 중 **5개**를 자동 추천하여 표시
- 각 키워드 칩에 도메인 색상 점(●) 표시
- 우측 상단 **[수정]** 버튼으로 편집 모드 진입 가능

### 편집 모드 (수정 버튼 클릭 시)

**선택 규칙**
| 조건 | 동작 |
|---|---|
| 3개 미만으로 [완료] 클릭 | 경고 메시지 표시, 편집 모드 유지 |
| 7개 이미 선택 | 나머지 칩 비활성화(회색), 추가 선택 차단 |
| 3~7개 선택 후 [완료] 클릭 | 뷰 모드로 전환 |

**카운터 상태**
| 조건 | 색상 |
|---|---|
| 3 ≤ 선택 수 ≤ 7 | 초록 (`#16A34A`) |
| 선택 수 < 3 | 빨강 (`#DC2626`) |

**편집 UI 구성**
```
최소 3개 · 최대 7개                      [N개 선택]
┌───────────────────────────────────────┐
│ [선택된 칩 미리보기] — × 버튼으로 제거   │
└───────────────────────────────────────┘
[강점 검색창]

E 실행력
성취  정리  신념  공정성  심사숙고  체계  집중  책임  복구

I 영향력
행동  주도력  커뮤니케이션  승부  최상화  자기확신  존재감  사교성

R 관계 구축
적응  연결성  개발  공감  화합  포용  개별화  긍정  절친

T 전략적 사고
분석  회고  미래지향  발상  수집  지적사고  배움  전략
```

### 갤럽 CliftonStrengths 34 도메인 분류

| 도메인 | 코드 | 색상 | 강점 목록 |
|---|---|---|---|
| 실행력 | E | 보라 `#6D28D9` | 성취, 정리, 신념, 공정성, 심사숙고, 체계, 집중, 책임, 복구 |
| 영향력 | I | 주황 `#C2410C` | 행동, 주도력, 커뮤니케이션, 승부, 최상화, 자기확신, 존재감, 사교성 |
| 관계 구축 | R | 파랑 `#1D4ED8` | 적응, 연결성, 개발, 공감, 화합, 포용, 개별화, 긍정, 절친 |
| 전략적 사고 | T | 초록 `#15803D` | 분석, 회고, 미래지향, 발상, 수집, 지적사고, 배움, 전략 |

### 강점별 정의 및 특성

AI가 인터뷰·이력서·강점 진단 결과를 분석하여 키워드를 추천할 때 아래 정의를 기준으로 사용한다.
각 강점은 **정의**, **기여하는 것**, **필요로 하는 것** 세 가지 속성으로 구성된다.

#### E — 실행력 (Executing)

| 강점 | 정의 | 기여하는 것 | 필요로 하는 것 |
|---|---|---|---|
| 성취 (Achiever) | 에너지가 왕성하며 열심히 일한다. 바쁘고 생산적인 데서 큰 만족감을 얻는다. | 강도 높고 지치지 않는 노력 | 자신의 속도대로 일할 자유 |
| 정리 (Arranger) | 조직하는 능력과 유연성을 함께 가진다. 생산성을 최대화하기 위해 모든 구성요소와 자원을 어떻게 조직할지 결정하는 것을 좋아한다. | 유연성과 상호작용 | 역동적인 환경 |
| 신념 (Belief) | 변하지 않는 확고한 핵심 가치를 가진다. 삶의 목적은 이러한 가치에서 나온다. | 가치 안정성, 명료함, 확신 | 삶의 이유 혹은 목적 |
| 공정성 (Consistency) | 사람들이 동등하게 취급되어야 한다는 점을 날카롭게 인식한다. 안정적인 루틴과 명료한 규칙, 절차를 갈망한다. | 예측 가능한 문화를 조성하는 규칙과 정책 | 표준적인 운영 절차 |
| 심사숙고 (Deliberative) | 의사결정 시 매우 신중하게 고려한다. 장애요소를 예측한다. | 철저하고 성실한 접근 | 의견을 내기 전에 듣고 생각할 시간 |
| 체계 (Discipline) | 정해진 절차와 구조를 즐긴다. 스스로 만든 질서로 세계를 설명한다. | 정확성과 세부 지향 | 구조화되고 질서 있는 환경 |
| 집중 (Focus) | 방향을 정해 일을 완수하며 방향에서 벗어나지 않기 위해 수정을 해낸다. 우선순위를 정한 다음 실행한다. | 집중과 방향을 통한 명료함 | 우선순위를 설정할 수 있게 해주는 목표 |
| 책임 (Responsibility) | 자신이 하겠다고 한 것에 대해 심리적 주인의식을 갖는다. 정직과 충직함 같은 안정적인 가치를 지키고자 한다. | 신뢰성과 충직함 | 주인의식을 발휘할 자유 |
| 복구 (Restorative) | 문제를 능숙하게 다룬다. 무엇이 잘못되었는지 파악하고 해결하는 데 뛰어나다. | 문제 있는 상황에 대한 용기와 창의성 | 반드시 해결해야 하는 문제들 |

#### I — 영향력 (Influencing)

| 강점 | 정의 | 기여하는 것 | 필요로 하는 것 |
|---|---|---|---|
| 행동 (Activator) | 머릿속 생각을 행동으로 옮겨 실현시킨다. 말만 하기보다는 당장 실행하기를 원한다. | 긴급성을 불어넣음 | 토론은 적게, 행동은 많이 |
| 주도력 (Command) | 존재감이 있다. 상황을 주도하고 의사결정을 내릴 수 있다. | 정서적 명료함 | 도전과 갈등 |
| 커뮤니케이션 (Communication) | 쉽게 자신의 생각을 말로 표현한다. 좋은 대화상대이자 발표에 능하다. | 사람들이 들어야 할 메시지에 주의를 집중시킴 | 생각을 논의할 상대, 청중 |
| 승부 (Competition) | 자신의 진전을 다른 사람의 성과와 비교해서 평가한다. 1등을 차지하기 위해 분투하며 경쟁을 즐긴다. | 최고가 되려는 포부 | 비교 대상 및 동기부여가 되는 동료 |
| 최상화 (Maximizer) | 개인이나 집단의 탁월성을 자극하기 위해 강점에 초점을 맞춘다. 좋은 것을 최상의 것으로 전환시킬 방법을 찾는다. | 품질 지향 | 수량만큼 품질도 중시되는 것 |
| 자기확신 (Self-Assurance) | 위험을 감수하고 삶을 이끌어가는 자신의 능력에 대한 자신감을 가진다. 내면의 나침반이 있다. | 필요한 위험을 기꺼이 감수하고자 하는 의지 | 단독적, 독립적으로 행동할 자유 |
| 존재감 (Significance) | 큰 영향을 미치기를 원한다. 독립적이며 조직과 주위 사람들에게 얼마나 영향을 미치는지를 고려한다. | 더 많이 원하고 생산하려는 열망 | 내가 최선을 다하도록 인정해주는 청중 |
| 사교성 (Woo) | 새로운 사람을 만나고 그들의 마음을 얻는 도전을 좋아한다. 어색한 분위기를 깨고 다른 사람과 연결되는 것에서 만족감을 얻는다. | 사람들이 모이는 자리에 활기를 불어넣음 | 다양한 사람들과의 만남 |

#### R — 관계 구축 (Relationship Building)

| 강점 | 정의 | 기여하는 것 | 필요로 하는 것 |
|---|---|---|---|
| 적응 (Adaptability) | 상황의 흐름에 따라 적응하는 것을 선호한다. '현재'에 충실한 경향이 있다. | 변화가 일어나는 방향으로 기꺼이 따라 감 | 즉각적으로 대응하는 현재의 압력 |
| 연결성 (Connectedness) | 모든 것들이 연결되어 있다고 믿는다. 우연이란 거의 존재하지 않으며 모든 일에 의미가 있다고 믿는다. | 알 수 없는 것에 대한 인정, 삶과 모든 존재에 대한 찬탄 | 자신보다 더 큰 무언가(가족, 팀, 글로벌 커뮤니티 등)의 일부가 되는 것 |
| 개발 (Developer) | 다른 사람의 잠재력을 알아보고 키워준다. 작은 진전의 신호를 알아차리며 사람들이 발전해가는 모습에서 만족을 얻는다. | 사람의 성장에 헌신할 시간과 에너지 | 투자할 대상자 |
| 공감 (Empathy) | 다른 사람의 삶이나 상황 속에 있는 것처럼 상상함으로써 다른 사람의 감정을 느낄 수 있다. | 감성 지능(Emotional Intelligence) | 울고 웃고, 감정을 발산할 자유 |
| 화합 (Harmony) | 합의를 추구한다. 갈등을 좋아하지 않고 갈등 대신 합의점을 찾는다. | 평화와 사랑, 갈등을 피하는 접근법 | 합의점, 공통된 의견 |
| 포용 (Includer) | 다른 사람을 받아들인다. 소외되었다고 느끼는 사람을 알아차리고 포용하기 위해 노력한다. | 다양성 수용에 대한 높은 수준의 관용 | 누구나 받아들일 수 있는 공간 |
| 개별화 (Individualization) | 각 사람이 지닌 독특한 개성에 흥미를 느낀다. 서로 다른 사람들이 어떻게 협력하여 생산적으로 일할 수 있는지 파악하는 데 재능이 있다. | 사람들의 가치 있는 배치에 대한 이해 | 각자에게 적합하게 설정된 개인적 기대 |
| 긍정 (Positivity) | 주위에 전염되기 쉬운 열정을 갖고 있다. 낙관적이며 사람들이 일하는 데 신이 나게 만든다. | 전염성 있는 에너지와 열정 | 기쁨과 삶의 드라마를 경험할 자유 |
| 절친 (Relator) | 다른 사람과 친밀한 관계를 즐긴다. 친밀한 사람들과 함께 목표를 달성하기 위해 열심히 일하는 데서 깊은 만족감을 얻는다. | 깊이 있고 투명한 관계 | 한 사람 한 사람과 상호작용할 시간과 기회 |

#### T — 전략적 사고 (Strategic Thinking)

| 강점 | 정의 | 기여하는 것 | 필요로 하는 것 |
|---|---|---|---|
| 분석 (Analytical) | 이유와 원인을 찾는다. 상황에 영향을 미칠 수 있는 모든 요인에 대해 생각하는 능력이 있다. | 감정적인 이슈에 대한 냉철한 사고 | 생각할 시간 |
| 회고 (Context) | 과거에 대해 생각하는 것을 즐긴다. 역사를 연구함으로써 현재를 이해한다. | 정확한 기억과 귀중한 기념물 | 토론과 결정을 위한 관련 배경 |
| 미래지향 (Futuristic) | 미래와 가능성에서 영감을 얻는다. 미래의 비전으로 사람들에게 영감을 준다. | 미리 그려 보기, 예상, 예측 | 예견되는 미래에 대해 말할 기회 |
| 발상 (Ideation) | 아이디어에 매료된다. 보기에는 상관없어 보이는 현상들의 연관성을 찾아낼 수 있다. | 새롭고 신선한 시각 | 통제나 제한 없이 가능성을 탐색할 자유 |
| 수집 (Input) | 수집하고 보관하고 싶어한다. 정보, 아이디어, 작품이나 심지어 관계도 축적할 수 있다. | 성장과 성과를 쉽게 해주는 실재적인 도구 | 자연스럽게 습득하는 자원을 보관할 공간 |
| 지적사고 (Intellection) | 지적인 활동을 하는 특징이 있다. 자기 성찰적이며 지적인 토론을 즐긴다. | 깊이 있는 이해와 지혜 | 성찰과 명상을 위한 시간 |
| 배움 (Learner) | 배움에 대한 강한 열망이 있으며 끊임없이 발전하고 싶어 한다. 배우는 과정 자체가 신나게 한다. | 배움의 관점과 배우는 과정에 대한 열정 | 새로운 정보와 경험을 접하기 |
| 전략 (Strategic) | 앞으로 나아가기 위한 대안들을 만들어낸다. 어떤 상황에서도 의미 있는 패턴과 이슈를 빠르게 파악해낸다. | 창의적인 예측, 상상, 끈기 | 중도에 궤도 수정할 수 있는 자유 |

## 기능 6 — AI 제안: 키워야 할 역량

### 표시 내용
역량명 + 우선순위 레이블 + 현재 수준 프로그레스 바

| 우선순위 | 색상 |
|---|---|
| 높음 | 파랑 (`#2D5BFF`) |
| 중간 | 노랑 (`#CA8A04`) |
| 낮음 | 회색 (`#A8A8A0`) |

### 역량 데이터 구조

```
역량명                우선순위    현재 수준
의사결정 & 실행력      높음        27%
커뮤니케이션 & 영향력  높음        41%
리더십 & 팀 빌딩       중간        37%
도메인 전문성 심화     낮음        57%
```

### 동작 흐름
```
뷰 모드 (기본)
  → [수정] 클릭
  → 역량명 편집 가능
  → 역량별 우선순위 선택 가능 (높음 / 중간 / 낮음)
  → [완료] 클릭
  → 뷰 모드로 전환
```

> **기획 의도**: 5년 후 목표(기능 5)와 현재 역량 수준 사이의 GAP을 기반으로 AI가 도출

## 기능 7 — 확정 버튼

| 상태 | 조건 | 스타일 |
|---|---|---|
| 활성 | 편집 모드 아님 + 키워드 3~7개 선택 완료 | 초록 (`#16A34A`) |
| 비활성 | 편집 모드 중 키워드 조건 미충족 | 회색, `cursor: not-allowed` |

### 동작 흐름
```
[확정하고 시작하기] 클릭
  → 편집 모드 중이면: 편집 완료 처리 후 유효성 재검사
  → 유효: 커리어 프로필 확정 → 역량 성장 대시보드로 이동
  → 무효(키워드 3개 미만): 경고 메시지 표시, 화면 유지
```
